# Next.js + Tailwind CSS Starter

Modern Next.js 15 boilerplate with Tailwind CSS, TypeScript, ESLint, and dark mode.

## Features

✨ TypeScript / Python type-safe by default
🎨 Modern styling with Tailwind / system fonts
⚡ Hot reload + fast iteration
🔒 Best practices applied (linting, formatting)
🌐 Production-ready deployment configs

## Quick Start

```bash
git clone https://github.com/1280itsuya/nextjs-tailwind-starter-20260610.git
cd nextjs-tailwind-starter-20260610
# Follow the README for setup
```

## Why this starter?

Most templates are bloated with vendor lock-in. This one focuses on the essentials,
so you can ship features instead of fighting with config.
